## 🖥 Run Locally

No build step needed. Just clone and open:

```bash

cd portfolio_new__
# Open index.html in your browser

start index.html       # Windows
xdg-open index.html   # Linux
```

Or use VS Code's **Live Server** extension for hot reload.
